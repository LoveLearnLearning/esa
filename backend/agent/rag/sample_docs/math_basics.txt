高等数学基础知识

第一章 极限与连续

1.1 极限的定义

极限是高等数学中最基础的概念之一。设函数 f(x) 在点 x0 的某个去心邻域内有定义，如果存在常数 A，对于任意给定的正数 ε（不论它多么小），总存在正数 δ，使得当 x 满足不等式 0 < |x - x0| < δ 时，对应的函数值 f(x) 都满足不等式 |f(x) - A| < ε，那么常数 A 就叫做函数 f(x) 当 x→x0 时的极限。

极限的存在性：如果极限存在，那么它一定是唯一的。这是极限的重要性质之一。

1.2 极限的四则运算

设 lim f(x) = A，lim g(x) = B，则：
1. lim [f(x) ± g(x)] = A ± B
2. lim [f(x) · g(x)] = A · B
3. lim [f(x) / g(x)] = A / B（其中 B ≠ 0）

这些运算规则大大简化了复杂极限的计算。

1.3 连续函数的定义

函数 f(x) 在点 x0 处连续需要满足三个条件：
1. f(x0) 有定义
2. lim f(x) 当 x→x0 时存在
3. lim f(x) = f(x0)

连续函数的性质：如果函数 f(x) 在闭区间 [a, b] 上连续，那么：
- f(x) 在 [a, b] 上有最大值和最小值
- f(x) 在 [a, b] 上一定取得最大值和最小值之间的任何值（介值定理）

第二章 导数与微分

2.1 导数的定义

导数是描述函数变化率的重要概念。设函数 y = f(x) 在点 x0 的某个邻域内有定义，当自变量 x 在 x0 处取得增量 Δx 时，相应地函数取得增量 Δy = f(x0 + Δx) - f(x0)。如果极限 lim(Δy/Δx) 当 Δx→0 时存在，则称函数 f(x) 在点 x0 处可导，该极限值称为 f(x) 在点 x0 处的导数。

导数的几何意义：函数 f(x) 在点 x0 处的导数等于曲线 y = f(x) 在点 (x0, f(x0)) 处的切线斜率。

2.2 基本求导公式

常用的求导公式包括：
- (c)' = 0（c 为常数）
- (x^n)' = n·x^(n-1)
- (sin x)' = cos x
- (cos x)' = -sin x
- (e^x)' = e^x
- (ln x)' = 1/x

这些公式是求导计算的基础，需要熟练掌握。

第三章 不定积分

3.1 不定积分的定义

如果在区间 I 上，可导函数 F(x) 的导函数为 f(x)，即对 I 上的任意 x 都有 F'(x) = f(x)，那么函数 F(x) 就称为 f(x) 在区间 I 上的一个原函数。

不定积分就是求原函数的过程。记作 ∫f(x)dx = F(x) + C，其中 C 为任意常数。

3.2 基本积分公式

常用的积分公式包括：
- ∫x^n dx = x^(n+1)/(n+1) + C（n ≠ -1）
- ∫(1/x) dx = ln|x| + C
- ∫e^x dx = e^x + C
- ∫sin x dx = -cos x + C
- ∫cos x dx = sin x + C

这些公式是积分计算的基础工具。
